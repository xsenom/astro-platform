# PDF banners

Положите баннер для PDF расчёта **«Уран в Близнецах»** по пути:

- `public/banners/uranus-gemini-pdf-banner.jpg`

После этого баннер будет доступен в приложении по URL:

- `/banners/uranus-gemini-pdf-banner.jpg`

Текущий рендер PDF для `uranus_gemini` отправляет этот URL в поле `banner_url`.
